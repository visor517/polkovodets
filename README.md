# Полководец
Военная браузерная игра, посвящённая эпохе Наполеоновских войн.
<img width="670" height="560" alt="001" src="https://github.com/user-attachments/assets/fd7ca179-68ee-41fc-a69a-3626a691bfed" />

## Описание
Игрок управляет армией (французской, русской или австрийской) на тактическом поле боя.  
Юниты передвигаются по шахматному принципу, учитывая тип войск.  
Важную роль играют командные радиусы генералов и маршалов.

### Технологии
- **Backend:** Django 5.2, DRF
- **Frontend:** JavaScript, Canvas

### Что сделано
- Игровой клиент на JavaScript + Canvas (поле 12×12, зум, панорамирование, подсветка возможных ходов)
- Сервер на Django + DRF (состояние игры, ходы, атаки, завершение игры)
- 6 типов юнитов с разными правилами движения и атаки
- игра восстанавливается по сохраненной сессии в куках

### Установка и запуск
```bash
# Клонировать репозиторий
git clone <url>
cd commander

# Создать виртуальное окружение
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Установить зависимости
pip install -r requirements.txt

# Миграции и запуск
python manage.py migrate
python manage.py runserver
```
